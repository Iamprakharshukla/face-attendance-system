🚀 QUICK START
╌╌╌╌╌╌╌╌╌╌╌╌╌

1. pip install -r requirements.txt
2. cp .env.example .env          (edit with your email)
3. python run.py
4. Open http://localhost:5000

🔧 TECHNOLOGY STACK

📂 KEY API ENDPOINTS
╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌

GET  /                           Home page
GET  /dashboard                  Attendance dashboard
GET  /register                   Registration page

GET  /camera/webcam              Webcam page
GET  /camera/ipcam               IP camera page
GET  /camera/stream              Live video stream
POST /camera/stop-stream         Stop stream

POST /upload/photo               Upload & analyze photo
POST /upload/video               Upload & analyze video

GET  /admin/known-people         List registered people
GET  /admin/today-attendance     Today's records
GET  /admin/attendance-report    Date range report
GET  /admin/export-attendance    Export CSV

POST /email/send-report          Send report email
POST /email/send-unknown-alert   Send alert email


⚙️ CONFIGURATION
╌╌╌╌╌╌╌╌╌╌╌╌

  - Distance threshold (0.6 default)
  - Upload folder path
  - Email settings
  - Session configuration
